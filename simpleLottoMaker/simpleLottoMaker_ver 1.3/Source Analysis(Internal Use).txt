prototpye 방식으로 만들었고 최종적으론 jQuery plugin 만들어서 호출해서 사용

[로또번호 추출]
6개의 배열을 생성
루프 돌려서 random한 숫자를 배열의 인덱스에 각각 넣고 
중복번호가 나오면 다시 루프를 돌리고 오름차순으로 정렬해서 화면에 뿌려줌
숫자이외의 영문, 한글, 특수문자등은 입력 못하게 입력 제한

[이미지롤링배너]
이미지롤링배너 원리는 여러가지 방식중에
jQuery의 css(), animate()를 이용해서
0번지 인덱스부터 마지막 이미지까지 이미지를 옆으로 이동시키고 옆으로 이동시킨 이미지는 투명도를 0으로 만들고 css에서 overflow: hidden처리(화면에서는 영역 밖으로 나간 이미지가 보이지 않음)하고
그 다음 인덱스의 이미지는 투명도를 1로 해서 보이게 함

setInterval()함수를 이용해서 타이머 시작(2초에 한번씩 롤링배너 동작 호출)

이미지에 mouseenter시 clearInterval() 함수 이용해서 이미지롤링 정지
mouseleave시 롤링 이미지 재시작

[시계 표시]
Date 객체 이용해서 년월일, 시, 분, 초 받아와서 가공 후 시계 영역에 표시
시간이 12시를 넘으면 pm으로, 12시 이전이면 am으로 표시
초가 50초 이상일 경우 적색으로 변환
0.5초마다 시간 출력
